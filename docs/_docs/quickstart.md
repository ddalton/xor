For the impatient, here's how to get a boilerplate XOR site up and running.
Add the following dependency to your pom.xml file.

```sh
<dependency>
    <groupId>tools.xor</groupId>
    <artifactId>xor</artifactId>
    <version>1.2.5</version>
</dependency>
```
