For the impatient, here's how to get XOR.
Add the following dependency to your Maven pom.xml file.

```sh
<dependency>
    <groupId>tools.xor</groupId>
    <artifactId>xor</artifactId>
    <version>1.7.25</version>
</dependency>
```
